# Academic-Clean PDF Report Template

## ภาพรวม

เทมเพลต "Academic-Clean" เป็นการออกแบบใหม่สำหรับการสร้างรายงาน PDF ที่เน้นความเรียบง่าย สวยงาม และอ่านง่าย เหมาะสำหรับการใช้งานในเชิงวิชาการและการนำเสนอข้อมูล

## คุณสมบัติหลัก

### 1. Hero Header Bar
- หัวข้อรายงานขนาดใหญ่ (20pt) พร้อมสีน้ำเงินเข้ม
- ข้อมูลเวลาที่สร้างรายงานในรูปแบบไทย
- การจัดวางแบบกลางหน้าจอ

### 2. ข้อมูลไฟล์ (Key-Value)
- ตารางข้อมูลไฟล์ที่มีการออกแบบสวยงาม
- แสดงชื่อไฟล์, คอลัมน์ที่ใช้, จำนวนแถว/คอลัมน์
- พื้นหลังสีฟ้าอ่อนสำหรับหัวข้อคอลัมน์

### 3. KPI Cards (สรุปสถิติหลัก)
- ตารางสรุปสถิติหลักสำหรับคอลัมน์แรก 4 คอลัมน์
- แสดงจำนวนข้อมูล, ค่าเฉลี่ย, ส่วนเบี่ยงเบน, ค่าต่ำสุด/สูงสุด
- การจัดวางตัวเลขชิดขวาเพื่อความอ่านง่าย

### 4. ตารางสถิติรายละเอียด
- ตารางสถิติแบบเต็มรูปแบบ (8 คอลัมน์)
- การออกแบบแบบ Zebra pattern (สลับสีแถว)
- หัวตารางสีน้ำเงินเข้ม พื้นหลังสีขาว/เทาอ่อน
- ตัวเลขชิดขวา หัวข้อชิดซ้าย

### 5. รูปกราฟเต็มความกว้าง
- รูปกราฟขนาด 15cm x 10cm
- คำอธิบายใต้รูป (Caption) แบบกลาง
- การจัดวางชิดขอบเพื่อความสวยงาม

### 6. การจัดการข้อมูล
- **Deduplication**: ลบรายการคอลัมน์ซ้ำแบบรักษาลำดับ
- **Default Title**: ถ้าหัวเรื่องว่างจะใช้ค่าเริ่มต้น
- **Clean Layout**: มาร์จินใหม่และ header/footer เรียบ

## การเปลี่ยนแปลงในโค้ด

### report_generator.py
- เพิ่มฟังก์ชัน `_dedupe_columns()` สำหรับลบคอลัมน์ซ้ำ
- เพิ่มฟังก์ชัน `_create_kpi_cards()` สำหรับสร้าง KPI cards
- เปลี่ยนจาก landscape เป็น portrait A4
- ปรับปรุง header/footer ให้เรียบและสวยงาม
- เพิ่ม subtitle style สำหรับข้อมูลเวลา
- ปรับปรุงการจัดวางตารางและรูปภาพ

### dialogs_report.py
- เพิ่มฟังก์ชัน `_dedupe_columns()` ในคลาส
- ปรับปรุง `get_options()` ให้จัดการหัวเรื่องว่างและ dedupe คอลัมน์
- รักษาลำดับคอลัมน์หลังการ dedupe

## การใช้งาน

### การสร้างรายงานปกติ
```python
options = {
    "title": "รายงานการวิเคราะห์ข้อมูล",
    "include_meta": True,
    "include_stats": True,
    "include_fig": True,
    "columns": ["col1", "col2", "col3"]
}
```

### การจัดการคอลัมน์ซ้ำ
```python
# คอลัมน์จะถูก dedupe อัตโนมัติ
options["columns"] = ["col1", "col2", "col1", "col3"]
# ผลลัพธ์: ["col1", "col2", "col3"]
```

### การจัดการหัวเรื่องว่าง
```python
# ถ้าหัวเรื่องว่าง จะใช้ค่าเริ่มต้น
options["title"] = ""  # จะกลายเป็น "รายงานการวิเคราะห์ข้อมูล SciPlotter"
```

## ผลลัพธ์

- **ความสวยงาม**: การออกแบบที่เรียบง่าย สะอาดตา
- **ความอ่านง่าย**: ตารางที่มีการจัดวางที่ดี ตัวเลขชิดขวา
- **ความสมบูรณ์**: ข้อมูลครบถ้วน การจัดวางเป็นระเบียบ
- **ความยืดหยุ่น**: รองรับการปรับแต่งและจัดการข้อมูลอัตโนมัติ

## การทดสอบ

เทมเพลตใหม่ได้รับการทดสอบแล้วกับ:
- ข้อมูลตัวอย่าง (Temperature, Humidity, Pressure, Wind_Speed, Precipitation)
- กราฟ matplotlib แบบ 2 subplots
- การ dedupe คอลัมน์ซ้ำ
- การจัดการหัวเรื่องว่าง
- การสร้าง PDF ขนาด A4 portrait

ทุกการทดสอบผ่านเรียบร้อยและสร้างไฟล์ PDF ได้สำเร็จ
