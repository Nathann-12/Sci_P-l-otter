# ฟีเจอร์ Create Derived Column

## ภาพรวม
ฟีเจอร์ "Create Derived Column" ช่วยให้ผู้ใช้สร้างคอลัมน์ใหม่จากนิพจน์ทางคณิตศาสตร์โดยใช้คอลัมน์ที่มีอยู่ใน DataFrame

## การเข้าถึง
- **เมนู**: Data → "สร้างคอลัมน์ใหม่…"
- **คีย์ลัด**: Ctrl+D

## วิธีใช้งาน

### 1. เปิด Dialog
- คลิกเมนู Data → "สร้างคอลัมน์ใหม่…" หรือกด Ctrl+D
- Dialog จะแสดงรายชื่อคอลัมน์ทั้งหมดที่มีอยู่ในข้อมูล

### 2. ตั้งชื่อคอลัมน์ใหม่
- ใส่ชื่อคอลัมน์ใหม่ในช่อง "ชื่อคอลัมน์ใหม่"
- ตัวอย่าง: `B_magnitude`, `Speed_kmh`, `Temperature_K`

### 3. เขียนนิพจน์
- ใช้ช่อง "นิพจน์" เพื่อเขียนสูตรทางคณิตศาสตร์
- อ้างอิงคอลัมน์ด้วย backtick: `` `Bx` ``, `` `Mag Field` ``
- คลิกสองครั้งที่คอลัมน์ในรายการเพื่อแทรกชื่อลงในนิพจน์

### 4. ใช้ฟังก์ชันลัด
- คลิกปุ่มฟังก์ชันเพื่อแทรก: `sqrt()`, `abs()`, `sin()`, `cos()`, `tan()`, `log()`, `exp()`, `min()`, `max()`, `mean()`

### 5. Preview และ Apply
- คลิก "🔍 Preview" เพื่อดูผลลัพธ์ 10 แถวแรก
- คลิก "✅ Apply" เพื่อสร้างคอลัมน์ใหม่ใน DataFrame

## ตัวอย่างนิพจน์

### นิพจน์พื้นฐาน
```
`Bx` * `By`                    # คูณสองคอลัมน์
`Bx` * `By` * `Bz`             # คูณสามคอลัมน์
`Speed` * 3.6                  # แปลงหน่วย m/s เป็น km/h
```

### คำนวณขนาดเวกเตอร์
```
sqrt(`Bx`**2 + `By`**2 + `Bz`**2)  # ขนาดของเวกเตอร์ B
sqrt(`Bx`**2 + `By`**2)            # ขนาดในระนาบ XY
```

### ฟังก์ชันคณิตศาสตร์
```
abs(`Signal`)                   # ค่าสัมบูรณ์
log(`Count` + 1)               # ลอการิทึมธรรมชาติ
sin(`Bx`)                      # ไซน์
cos(`By`)                      # โคไซน์
exp(`Temperature` / 100)       # เอกซ์โพเนนเชียล
```

### ฟังก์ชันสถิติ
```
maximum(`Bx`, `By`)            # ค่าสูงสุดระหว่างสองคอลัมน์
minimum(`Temperature`, `Pressure` / 100)  # ค่าต่ำสุด
mean(`Signal`)                 # ค่าเฉลี่ย
```

### นิพจน์ซับซ้อน
```
sqrt(`Bx`**2 + `By`**2) * `Speed`  # ขนาดเวกเตอร์คูณความเร็ว
`Temperature` * `Pressure` / 1000   # อุณหภูมิคูณความดัน
```

### ชื่อคอลัมน์ที่มีช่องว่าง
```
`Mag Field` * 2                # คอลัมน์ที่มีช่องว่าง
`Temperature (°C)` + 273.15    # คอลัมน์ที่มีอักขระพิเศษ
```

## ฟังก์ชันที่รองรับ

### ฟังก์ชันคณิตศาสตร์
- `sqrt(x)` - รากที่สอง
- `abs(x)` - ค่าสัมบูรณ์
- `sin(x)` - ไซน์
- `cos(x)` - โคไซน์
- `tan(x)` - แทนเจนต์
- `log(x)` - ลอการิทึมธรรมชาติ
- `exp(x)` - เอกซ์โพเนนเชียล

### ฟังก์ชันสถิติ
- `min(x)` - ค่าต่ำสุด
- `max(x)` - ค่าสูงสุด
- `mean(x)` - ค่าเฉลี่ย
- `std(x)` - ส่วนเบี่ยงเบนมาตรฐาน
- `minimum(x, y)` - ค่าต่ำสุดระหว่างสองค่า
- `maximum(x, y)` - ค่าสูงสุดระหว่างสองค่า

## ข้อกำหนดและข้อจำกัด

### ข้อกำหนด
- ต้องมีข้อมูลใน DataFrame ก่อน
- ชื่อคอลัมน์ใหม่ต้องไม่ซ้ำกับที่มีอยู่ (หรือยืนยันการแทนที่)
- นิพจน์ต้องใช้ไวยากรณ์ที่ถูกต้อง

### ข้อจำกัด
- รองรับเฉพาะฟังก์ชันที่ปลอดภัย (ไม่รองรับฟังก์ชันที่อาจเป็นอันตราย)
- ไม่รองรับการอ้างอิงคอลัมน์แบบ dynamic หรือ conditional
- การประเมินใช้ pandas.eval() หรือ Python eval() ขึ้นอยู่กับความซับซ้อน

## การจัดการข้อผิดพลาด

### ข้อผิดพลาดทั่วไป
- **"ไม่พบคอลัมน์"**: ตรวจสอบชื่อคอลัมน์ในนิพจน์
- **"นิพจน์มีไวยากรณ์ไม่ถูกต้อง"**: ตรวจสอบไวยากรณ์ของนิพจน์
- **"นิพจน์มีประเภทข้อมูลที่ไม่เข้ากัน"**: ตรวจสอบประเภทข้อมูลของคอลัมน์

### การจัดการ NaN และ Inf
- ค่า inf/-inf จะถูกแปลงเป็น NaN อัตโนมัติ
- ค่า NaN จะแสดงในพรีวิวและผลลัพธ์
- ระบบจะเตือนเมื่อพบค่า NaN จำนวนมาก

## เทคนิคการใช้งาน

### 1. การแปลงหน่วย
```
`Speed` * 3.6                    # m/s → km/h
`Temperature` + 273.15           # °C → K
`Pressure` / 1000               # Pa → kPa
```

### 2. การคำนวณทางฟิสิกส์
```
sqrt(`Bx`**2 + `By`**2 + `Bz`**2)  # ขนาดสนามแม่เหล็ก
`Mass` * `Acceleration`            # แรง F = ma
`Voltage` / `Current`             # ความต้านทาน R = V/I
```

### 3. การประมวลผลสัญญาณ
```
abs(`Signal`)                     # ค่าสัมบูรณ์ของสัญญาณ
log(`Count` + 1)                  # ลอการิทึมเพื่อลด skewness
sin(`Frequency` * 2 * pi * `Time`) # สัญญาณไซน์
```

## การแก้ไขปัญหา

### ปัญหาที่พบบ่อย
1. **นิพจน์ไม่ทำงาน**: ตรวจสอบชื่อคอลัมน์และไวยากรณ์
2. **ผลลัพธ์เป็น NaN**: ตรวจสอบการหารศูนย์หรือค่าที่ไม่ถูกต้อง
3. **Dialog ไม่เปิด**: ตรวจสอบว่ามีข้อมูลใน DataFrame หรือไม่

### การ Debug
- ใช้ Preview เพื่อดูผลลัพธ์ก่อน Apply
- ตรวจสอบสถิติใน tooltip ของตารางพรีวิว
- ลองใช้นิพจน์ง่ายๆ ก่อน แล้วค่อยเพิ่มความซับซ้อน

## การอัปเดตและพัฒนาต่อ

### แผนการพัฒนาต่อ
- เพิ่มฟังก์ชันทางสถิติเพิ่มเติม
- รองรับการอ้างอิงคอลัมน์แบบ conditional
- เพิ่มการตรวจสอบหน่วยและมิติ
- รองรับการสร้างคอลัมน์หลายคอลัมน์พร้อมกัน

### การรายงานปัญหา
หากพบปัญหา กรุณารายงานพร้อมข้อมูล:
- นิพจน์ที่ใช้
- ข้อมูลตัวอย่าง
- ข้อความผิดพลาดที่แสดง
