# CDF File Reading Troubleshooting Guide

## ปัญหาที่พบ
โปรแกรมไม่สามารถอ่านไฟล์ CDF ได้

## สาเหตุและวิธีแก้ไข

### 1. การติดตั้ง Package
**ปัญหา**: ไม่มี package `cdflib` หรือเวอร์ชันเก่าเกินไป

**วิธีแก้ไข**:
```bash
pip install cdflib>=1.2
pip install xarray>=2024.1.0
pip install netCDF4>=1.6.0
```

### 2. การจัดการ Error ที่ดีขึ้น
**ปัญหา**: โปรแกรม crash เมื่ออ่านไฟล์ CDF ไม่สำเร็จ

**วิธีแก้ไข**: 
- เพิ่มฟังก์ชัน fallback 3 วิธีในการอ่านไฟล์ CDF
- แสดงข้อความ error ที่ชัดเจน
- จัดการ exception อย่างเหมาะสม

### 3. ฟังก์ชัน Fallback สำหรับการอ่าน CDF

#### วิธีที่ 1: Heuristic (แนะนำ)
- ใช้ `cdflib` โดยตรง
- ค้นหาตัวแปรเวลาอัตโนมัติ
- เลือกตัวแปรข้อมูลที่เหมาะสม

#### วิธีที่ 2: xarray
- ใช้ `cdflib.cdf_to_xarray()`
- แปลงเป็น DataFrame
- เหมาะสำหรับไฟล์ CDF ที่ซับซ้อน

#### วิธีที่ 3: Simple
- ใช้ `cdflib` พื้นฐาน
- อ่านตัวแปรทั้งหมด
- แปลงเวลาเป็น datetime

### 4. การทดสอบ
รันสคริปต์ทดสอบเพื่อตรวจสอบการทำงาน:
```bash
python test_cdf_simple.py
```

### 5. การใช้งาน
1. เปิดแอปพลิเคชัน: `python main.py`
2. เลือกไฟล์ .cdf
3. โปรแกรมจะลองอ่านด้วยวิธีทั้ง 3 วิธี
4. หากสำเร็จ จะแสดงข้อมูลในตาราง

## ข้อควรระวัง
- ไฟล์ CDF ต้องมีตัวแปรเวลาที่แปลงเป็น datetime ได้
- ไฟล์ CDF ต้องมีตัวแปรข้อมูลที่เป็นตัวเลข
- ขนาดข้อมูลต้องสอดคล้องกัน

## การรายงานปัญหา
หากยังมีปัญหา กรุณาแจ้ง:
1. ข้อความ error ที่แสดง
2. เวอร์ชันของ cdflib, numpy, pandas
3. ตัวอย่างไฟล์ CDF ที่มีปัญหา
